Portfolio Project

This is a little project for my portfolio using a REST API . You can choose between four methods of loading Countries and information about them.

Region

The Region button opens a menu with five options. Every option will load only the countries from it's region

All Countries

On click this button loads all the countries from the API's database.

My Location

The My Location button displays the country in which you are situated right now.(Run the project with https in Chrome)

Search Country

You can also fill out the form to show a country you wish to see.

Learn More

Once a country is loaded, you can click on the Learn More button. It will display a modal window with information about the country.
